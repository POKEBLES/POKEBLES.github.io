## 📋 Common Tasks

### I Want To...

**→ See the website right now**
1. Open [QUICKSTART.md](QUICKSTART.md)
2. Follow "Step 2: Start a Local Server"
3. Open http://localhost:8000

**→ Change colors**
1. Open `css/style.css`
2. Edit lines 8-18 (CSS variables)
3. Save and refresh browser

**→ Add my logo**
1. Put logo file in `images/logo.png`
2. Refresh browser

**→ Update contact info**
1. Search for "Bacood, Sta. Mesa" in all HTML files
2. Replace with your address
3. Do same for phone, email, etc.

**→ Add my portfolio projects**
1. Add images to `images/portfolio/`
2. Edit `portfolio.html` (line 80+)
3. Update project cards with your info

**→ Connect forms to email**
1. Read [SETUP.md](SETUP.md) "Email Service Setup"
2. Choose option (EmailJS, FormSpree, or custom)
3. Follow integration steps

**→ Deploy to web**
1. Read [SETUP.md](SETUP.md) "Deployment" section
2. Choose hosting (Netlify, Vercel, or traditional)
3. Follow deployment steps

**→ Add Google Maps**
1. Get API key from Google Cloud Console
2. Read [SETUP.md](SETUP.md) "Google Maps Integration"
3. Add script tag to contact.html
4. Update coordinates in js/contact.js

**→ Implement real authentication**
1. Read [SETUP.md](SETUP.md) "Backend Integration"
2. Create API endpoints for login/register
3. Update functions in `js/auth.js`
4. Connect to your database
