## 🛠️ CSS Variables for Future Customization

If you ever need to adjust colors, edit these in `css/style.css`:

```css
:root {
    --primary-color: #d63946;      /* Main brand red */
    --primary-dark: #a41a28;       /* Dark red hover */
    --primary-light: #e85a6a;      /* Light red accent */
    --secondary-color: #f77f00;    /* Orange accent */
    --secondary-dark: #e07000;     /* Dark orange */
    --accent-color: #ffc93c;       /* Gold highlight */
    --teal-color: #20b2aa;         /* Teal accent */
    /* ... neutral colors ... */
}
```

All colors throughout the site reference these variables, so changes propagate automatically!
